# 蘇格拉底式對話 Workflow

## 方法

蘇格拉底式對話是一種透過提問引導思考的方法，目的不是給予答案，而是幫助對方發現自己的想法。

## 提問類型

### 1. 澄清問題
- 「你說的 X 是什麼意思？」
- 「可以舉個例子嗎？」
- 「這跟 Y 有什麼關係？」

### 2. 探究假設
- 「你為什麼認為這是真的？」
- 「這個假設的基礎是什麼？」
- 「如果這個假設不成立呢？」

### 3. 尋求證據
- 「有什麼支持這個觀點？」
- 「這個證據可靠嗎？」
- 「有反面的證據嗎？」

### 4. 考慮替代觀點
- 「有沒有其他可能的解釋？」
- 「如果從另一個角度看呢？」
- 「反對者會怎麼說？」

### 5. 探索後果
- 「如果這是真的，會導致什麼？」
- 「這個觀點的實際影響是什麼？」
- 「長期來看會如何？」

### 6. 質疑問題本身
- 「為什麼這個問題重要？」
- 「這是正確的問題嗎？」
- 「還有什麼我們該問的？」

## 對話原則

1. **謙遜**：承認自己不知道
2. **好奇**：真誠地想理解
3. **耐心**：給對方思考時間
4. **尊重**：不評判，只探問
5. **開放**：願意改變自己的想法
