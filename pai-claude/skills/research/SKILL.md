# Research Skill

技術調研和資料收集的專業技能。

## Workflow Routing

- 深度研究 → [workflows/deep-research.md](workflows/deep-research.md)

## 研究方法

### 1. 定義問題
- 明確研究目標
- 確定評估標準
- 設定時間範圍

### 2. 資料收集
- 官方文檔
- GitHub 專案
- 技術部落格
- 論文/報告

### 3. 分析比較
- 建立比較矩陣
- 權衡優缺點
- 考慮長期維護

### 4. 結論建議
- 提供明確建議
- 說明理由
- 列出風險

## 輸出格式

```markdown
## 研究主題：[主題]

### 背景
[為什麼需要研究這個問題]

### 選項分析
| 選項 | 優點 | 缺點 | 適用場景 |
|------|------|------|----------|
| A    | ...  | ...  | ...      |
| B    | ...  | ...  | ...      |

### 建議
[最終建議及理由]

### 參考資料
- [連結1]
- [連結2]
```
