# 股票分析 Workflow

## 分析流程

### 1. 公司概況

- 公司名稱、代號
- 產業類別
- 主要業務
- 競爭地位

### 2. 財務分析

#### 獲利能力
- 毛利率
- 營業利益率
- 淨利率
- ROE / ROA

#### 成長性
- 營收 YoY
- 獲利 YoY
- 股息成長

#### 財務健康
- 負債比率
- 流動比率
- 利息保障倍數

#### 現金流
- 營運現金流
- 自由現金流
- 資本支出

### 3. 估值分析

- 本益比 (P/E)
- 股價淨值比 (P/B)
- EV/EBITDA
- DCF 估值（如適用）

### 4. 風險評估

- 產業風險
- 競爭風險
- 財務風險
- 總體經濟風險

## 輸出格式

```markdown
# [公司名稱] ([代號]) 分析

## 基本資料
- 產業：
- 市值：
- 股價：

## 財務摘要
| 指標 | 數值 | 同業平均 |
|------|------|----------|
| ...  | ...  | ...      |

## 投資論點
### 優勢
### 風險
### 估值

## 結論
[總結評估]
```
