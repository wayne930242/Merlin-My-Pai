# 投資組合檢視 Workflow

## 檢視項目

### 1. 資產配置

- 股票 vs 債券 vs 現金
- 地區分布
- 產業分布
- 個股權重

### 2. 績效分析

- 整體報酬率
- 個別持股報酬
- 與基準比較
- 風險調整報酬

### 3. 風險評估

- 集中度風險
- 相關性分析
- 下檔風險

### 4. 再平衡需求

- 偏離目標配置
- 調整建議

