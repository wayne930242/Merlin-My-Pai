# TRPG Skill

桌上角色扮演遊戲的專業輔助技能。

## Workflow Routing

- 角色創建 → [workflows/character-creation.md](workflows/character-creation.md)
- 故事生成 → [workflows/story-generation.md](workflows/story-generation.md)

## 支援系統

- D&D 5e
- Pathfinder 2e
- Call of Cthulhu
- 自創系統

## 功能

### 角色相關
- 角色背景生成
- 人格特質建議
- 成長弧線規劃

### 故事相關
- 劇情鉤子生成
- NPC 快速創建
- 場景描述
- 隨機事件表

### DM 工具
- 遭遇設計
- 獎勵平衡
- 即興應對建議
