# 角色創建 Workflow

## 創建流程

### 1. 基本概念

回答以下問題：
- 這個角色的核心概念是什麼？（一句話描述）
- 什麼驅動這個角色？（動機）
- 這個角色害怕什麼？（弱點）

### 2. 背景故事

#### 出身
- 家庭背景
- 成長環境
- 重要經歷

#### 轉折點
- 什麼事件改變了角色的人生？
- 為什麼踏上冒險之路？

#### 關係
- 盟友/朋友
- 敵人/對手
- 重要 NPC

### 3. 人格特質

- **理想**：角色相信什麼？
- **羈絆**：角色珍視什麼？
- **缺陷**：角色的弱點是什麼？

### 4. 外觀描述

- 外表特徵
- 穿著風格
- 特殊標記

## 快速生成器

```
角色名稱：[名字]
種族/職業：[種族] [職業]
一句話概念：[核心概念]

背景鉤子：
- [重要過去事件]
- [未解決的問題]
- [個人目標]

人格：
- 理想：[信念]
- 羈絆：[珍視的人事物]
- 缺陷：[性格弱點]
```
