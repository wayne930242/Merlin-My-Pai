# 故事生成 Workflow

## 劇情鉤子生成

### 類型

1. **任務型**：有人委託完成某事
2. **發現型**：意外發現線索或秘密
3. **危機型**：緊急事件需要處理
4. **個人型**：與角色背景相關

### 快速生成器

```
[誰] 需要 [什麼]，因為 [原因]，
但是 [障礙]，而且 [時間壓力]。
```

## NPC 快速創建

```
名字：[名字]
職業：[職業]
特徵：[一個明顯特徵]
動機：[想要什麼]
秘密：[隱藏的事情]
說話方式：[語氣/口頭禪]
```

## 場景描述框架

### 五感描述
- **視覺**：看到什麼？
- **聽覺**：聽到什麼？
- **嗅覺**：聞到什麼？
- **觸覺**：感受到什麼？
- **氛圍**：整體感覺？

### 重要元素
- 可互動物品
- 隱藏細節
- 潛在危險

## 即興表格

### 隨機併發症 (d6)
1. 有人在監視
2. 時間比預期緊迫
3. 目標已經改變
4. 意外的盟友出現
5. 環境發生變化
6. 發現額外的秘密
